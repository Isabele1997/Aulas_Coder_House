Sintaxe: CSS
setor{propriedade1:valor;} ex: h1{color:red}

*Atributo Class:* É uma lista das classes de um elemento separada por espaços. Permite que o CSS 
e o Javascript selecione e acesse elementos específicos.
É utilizado para aplicar o mesmo estilo para mais de um elemento.
HTML: Atributo class="" ex: <img src=""class="bordasarredendadas"/>
CSS: bordasarredendadas{/*código CSS*/}

*Atributo ID*: Geralmente usado para nomear partes do código e setores.
HTML: <section id="nome"></section>
CSS: #nome{/*códico CSS*/}

*Herança:* Em geral estas propriedades são intuitivas, ex: você pode herdar o tamanho e a color
da letra de um elemento pai, a menos que o elemento filho tenha outros estilos aplicados.
div{color:red} ou <div> <p> Paragrafo aqui </p> </div>

*Estilização para listas:* list-style-type ex: ol{list-style-type:none}
